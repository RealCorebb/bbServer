# 关于香橙派：  
你需要启用I2C和SPI  

# 关于Adafruit Blinka：  
这是能够让很多Linux开发板能运行CircuiPython的库，如果装的是armbian系统能够直接使用，如果是其它系统，它需要识别板子型号，需要/etc/armbian-release 这个文件。通常在/etc文件夹内会有像是：/etc/orangepi-release，把这个文件重命名即可。
